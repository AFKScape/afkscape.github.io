AFKScape - A resizable monitor for Runescape©, for Windows Vista, 7, 8, or 10.

The Latest version [0.0.0.2] is here: ZZZZZ

Allows you to clone any of your Runescape© clients and keep it *always-on-top* while working with other windows.

“Click forwarding”: allows interaction with the Runescape© client and replicate dragging.

5 minute log-out timer with colour notification to prevent lobbying.

Hidable menu bar so the user can use AFKS for more than just Runescape.

Remember to check the read-me before use. Use AFKS at your own risk!

Thank you for testing out AFKS.
